SafeRoute 🚦
📌 Problem Statement

Most existing navigation systems focus only on the fastest or shortest route, ignoring critical safety factors such as:

Accident-prone zones
Poorly lit streets
Unsafe or high-risk areas

This can put users at risk, especially during night travel or in unfamiliar locations.

⚙️ How to Run Locally: 
Clone the Repository
git clone https://github.com/your-username/saferoute.git
cd saferoute

💡 Solution

SafeRoute enhances traditional navigation by incorporating safety metrics into route planning.

It analyzes multiple data sources to provide:

🟢 Safest Route (least risk)
🔵 Fastest Route (minimum time)
🔴 Danger Zones highlighted on the map
🛡️ Safety Score for each route

Features
📍 Select start and destination directly on the map
🛣️ Displays multiple route options (fastest vs safest)
🔴 Visual representation of accident-prone / unsafe areas
🛡️ Route safety scoring system
🔵 Color-coded routes for easy comparison
🚗 Displays multiple transport modes
⌛ Predictive time modes
📊 Area Statistics 
🆘 SOS 

🛠️ Tech Stack

Full Stack: 
Leaflet.js (for interactive maps)
HTML (CSS, JavaScript)

APIs & Data Sources:
OpenStreetMap
Overpass API
Geocoding APIs
